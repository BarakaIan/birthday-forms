Here is our landing page where you get to fill in a form which displays your birthday, gender and as well as your Akan name.

The application checks whether the date and month entered is valid. if either of them is invalid, the user is alerted and asked to enter a valid one. The application outputs to the user, their Akan name depending on their gender. For example, if a user is male and the result of the calculation is 0, then the application should match that with Kwasi since Kwasi corresponds to Sunday which has an index 0. Hence the output the user will see should be something like “Your Akan name is Kwasi”. That is all that we do.
